# 🔒 Winlocker Simulation with Math Unlock (Educational Tool)

![License: MIT](https://shields.io)
![Language: Python](https://shields.io)
![OS: Windows](https://shields.io)

Скрипт на Python и Tkinter, симулирующий блокировщик экрана (Winlocker) с уникальной механикой **математической разблокировки**. Разработан для демонстрации механизмов перехвата ввода, модификации реестра Windows и создания устойчивого полноэкранного UI в целях исследования киберугроз.

⚠️ **ВНИМАНИЕ: Проект создан строго в образовательных целях и для легитимного тестирования систем безопасности. Автор не несет ответственности за вред, причиненный использованием данного кода.**

---

## 🧮 Механика разблокировки (Math Unlock System)
В отличие от классических блокировщиков, требующих фиксированный статический пароль, данный инструмент использует динамическую верификацию:
* Система генерирует случайный математический пример (сложение или вычитание двузначных чисел).
* Доступ к операционной системе возвращается **только после ввода правильного ответа**.
* При неверном вводе интерфейс выводит предупреждение, сбрасывает поле ввода и обновляет статус.

---

## 🚀 Функционал (Features)
* 🖥️ **Полноэкранный режим** — перекрытие UI в режиме `fullscreen` поверх всех окон с подменой курсора на пиратский флаг (`pirate`).
* 🔒 **Защита от закрытия** — жесткая блокировка системных комбинаций клавиш через библиотеку `keyboard` (доступен ввод только цифр и Backspace) и программное отключение Диспетчера задач Windows.
* ⚡ **Административные привилегии** — автоматический запрос повышенных прав (`IsUserAnAdmin`/`ShellExecuteW`) для легитимного внесения изменений в систему.
* ⚙️ **Мониторинг ресурсов** — динамический вывод текущей загрузки ЦП и ОЗУ в реальном времени с помощью `psutil`.
* 🎼 **Звуковой фон** — изолированный запуск аудиофайла в отдельном процессе через `multiprocessing` для предотвращения фризов интерфейса.

## 💻 Установка и запуск (Installation & Usage)

> **Важно:** Запуск производите только в изолированной среде (виртуальная машина или Sandbox). Скрипт активно модифицирует ветку реестра `HKCU` и прописывает себя в автозагрузку.

1. Клонируйте репозиторий:
   ```bash
   git clone github.com
   ```
2. Перейдите в каталог проекта:
   ```bash
   cd winlocker
   ```
3. Установите необходимые внешние библиотеки:
   ```bash
   pip install keyboard psutil playsound
   ```
4. Поместите аудиофайл в директорию (опционально) и укажите его имя в вызове `playsound()`.
5. Запустите симуляцию (автоматически запросит права Администратора):
   ```bash
   python main.py
   ```

## 🗂️ Архитектура кода (Code Overview)
* `generate_math_problem()` — отвечает за генерацию случайных чисел, выбор математического оператора и расчет эталонного ответа для разблокировки.
* `set_registry_settings(disable=True/False)` — управляет системным ключом `DisableTaskMgr` и параметром автозагрузки `SystemBlocker`.
* `block_keys()` — перехватывает скан-коды клавиатуры, фильтруя любые нажатия, кроме математического ввода.
* `window()` — инициализирует бесконечный цикл Tkinter, блокирует стандартную кнопку закрытия окна, запускает таймер обратного отсчета и анимацию заголовка.

## 📦 Сборка в исполняемый файл (Compiling to .exe)

Вы можете скомпилировать скрипт в один автономный `.exe` файл, чтобы запускать его без установленного Python. Для этого используется утилита `pyinstaller`.

### Шаг 1. Установка компилятора
Установите `pyinstaller` через терминал:
```bash
pip install pyinstaller
```

### Шаг 2. Запуск сборки
Выполните команду в корневой папке проекта:
```bash
pyinstaller --onefile --noconsole --uac-admin main.py
```

* **Флаги сборщика:**
  * `--onefile` — собирает все зависимости в один единственный `.exe` файл.
  * `--noconsole` — скрывает стандартное черное консольное окно Windows при запуске (остается только окно Tkinter).
  * `--uac-admin` — автоматически добавляет манифест запроса прав Администратора (при запуске файла Windows сразу потребует подтверждение UAC).

### Шаг 3. Результат
После завершения процесса в папке проекта появятся новые каталоги. Готовый файл `main.exe` будет находиться в папке **`dist/`**.

> ⚠️ **Примечание:** Собранный `.exe` файл может вызывать ложные срабатывания встроенного Windows Defender или сторонних антивирусов (так как код использует хуки клавиатуры и правит реестр). Для тестов добавляйте файл в исключения антивируса.
