# 🔒 Winlocker Simulation (Educational Tool)

![License: MIT](https://shields.io)
![Language: Python](https://shields.io)
![OS: Windows](https://shields.io)

Скрипт на Python и Tkinter, симулирующий блокировщик экрана (Winlocker). Разработан для демонстрации механизмов перехвата ввода, модификации реестра и создания устойчивого полноэкранного UI в целях исследования киберугроз.

⚠️ **ВНИМАНИЕ: Проект создан строго в образовательных целях и для легитимного тестирования систем безопасности. Автор не несет ответственности за вред, причиненный использованием данного кода.**

---

## 🚀 Функционал (Features)
* 🖥️ **Полноэкранный режим** — перекрытие UI в режиме `fullscreen` поверх всех окон с подменой курсора (`pirate`).
* 🔒 **Защита от закрытия** — блокировка комбинаций клавиш через библиотеку `keyboard` (кроме цифр и Backspace) и отключение Диспетчера задач.
* ⚡ **Административные привилегии** — автоматический запрос прав Администратора (`IsUserAnAdmin`/`ShellExecuteW`) для внесения изменений.
* 🗝️ **Динамическая разблокировка** — снятие ограничений происходит только после решения математического примера случайной генерации.
* ⚙️ **Мониторинг ресурсов** — вывод системных данных (ЦП, ОЗУ) в реальном времени с помощью `psutil`.
* 🎼 **Звуковой фон** — изолированный запуск аудиофайла в отдельном процессе `multiprocessing`.

## 💻 Установка и запуск (Installation & Usage)

> **Важно:** Запуск производите только в изолированной среде (виртуальная машина / Sandbox). Скрипт вносит изменения в ветку реестра `HKCU`.

1. Клонируйте репозиторий:
   ```bash
   git clone github.com
   ```
2. Перейдите в каталог проекта:
   ```bash
   cd winlocker
   ```
3. Установите необходимые библиотеки:
   ```bash
   pip install keyboard psutil playsound
   ```
4. Поместите аудиофайл в директорию (опционально) и укажите имя в `playsound()`.
5. Запустите симуляцию от имени Администратора:
   ```bash
   python main.py
   ```

## 🗂️ Архитектура кода (Code Overview)
* `set_registry_settings(disable=True/False)` — управляет ключом `DisableTaskMgr` и параметром автозагрузки `SystemBlocker`.
* `block_keys()` — перехватывает скан-коды клавиатуры, предотвращая использование системных сочетаний.
* `window()` — инициализирует бесконечный цикл Tkinter, вешает хук на закрытие окна, запускает таймер обратного отсчета и анимацию текста.
